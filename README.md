Data link: https://www.kaggle.com/datasets/orvile/sperm-morphology-image-data-set-smids/data

* 70% of data for training, 20% for validation and 10% for testing

  - cuda 11.8
  - cudnn 8.9.7
  - TensorRT 8.6.1
  - python 3.10
  - tensorflow 2.13.1
  - tensorflow-hub 0.15.0
  - numpy 1.24.3
  - pandas 2.1.4
  - scikit-learn 1.3.2
